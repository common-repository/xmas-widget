== Installation ==

This section describes how to install the plugin and get it working.

1. Visit 'Plugins > Add New'
2. Upload xmas-widget-little-elf.zip
3. Activate Xmas-Widget from your Plugins page.
4. That's all! Your wordpress is now ready for Xmas!
